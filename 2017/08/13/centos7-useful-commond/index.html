centos7常用命令

发表于 2017-08-13 | 分类于 centos7

查看端口占用程序并结束

查询nginx主进程号

ps -ef | grep nginx

  • 停止进程
    kill -QUIT 11111 #主进程号
  • 快速停止
    kill -TERM 11111 #主进程号
  • 强制停止
    pkill -9 nginx

常用zip相关命令

  • 安装zip/unzip

  • 压缩文件/目录
    例如:mydata目录压缩为mydata.zip

  • 解压
    -d指定目录，缺省为当前目录

显示当前所在目录

pwd

查看所有java进程

杀死所有Java进程

ps -ef|grep 'java' |grep -v grep|cut -c 9-15|xargs kill -9 >/dev/null 2>&1
或者
pkill -9 java
